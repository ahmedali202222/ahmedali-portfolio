## ğŸ“¦ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/ahmedali202222/ahmedali-portfolio.git
   cd ahmedali-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```

6. **Deploy to GitHub Pages**
   ```bash
   npm run deploy
   ```
